# Fegeya Gretea
## Gretea (aka green tea), new generation programming language.

### A taste of Gretea's syntax:
```rust
import tea.green.fmt

module hello {
    fn hello#display_it<What>(what: What) {
        fmt#println(what)
    }
}

fn main() = int {
  hello#display_it("Hi Gretea!")

  . 0
}
```
